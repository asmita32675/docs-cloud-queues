Document change history
~~~~~~~~~~~~~~~~~~~~~~~~~

This version of the document replaces and obsoletes all earlier
versions. The most recent changes are described in the following table.

March 4, 2015

Removed the London endpoint, since Rackspace now has one global
endpoint for authentication using the Rackspace Cloud Identity
service. See `Authentication <Authentication-d1e647.html>`__.

January 19, 2015

-  Corrected `Limits <Absolute-Limits-d1e003.html>`__ to show the number
   of messages that can be claimed at one time as 1 - 100.

-  Updated the information about the ``limit`` parameter in the claim
   messages operation description and provided examples.
